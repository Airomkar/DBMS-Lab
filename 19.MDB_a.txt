| **Q No.** | **Oral Question**                                        | **Answer**                                                                                             |
| --------- | -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| 1         | What is MongoDB?                                         | MongoDB is a NoSQL database that stores data in JSON-like documents called BSON.                       |
| 2         | What is a Collection in MongoDB?                         | A collection is similar to a table in SQL; it stores multiple documents.                               |
| 3         | What is a Document in MongoDB?                           | A document is a single record in MongoDB represented in JSON format.                                   |
| 4         | Difference between insertOne() and insertMany()?         | `insertOne()` inserts one document, while `insertMany()` inserts multiple documents at once.           |
| 5         | What is pretty() in MongoDB?                             | It displays documents in a more readable, formatted style.                                             |
| 6         | What does `$gte` mean?                                   | `$gte` means "greater than or equal to" in MongoDB query conditions.                                   |
| 7         | What is `$in` operator used for?                         | `$in` matches any of the values specified in an array (like OR condition).                             |
| 8         | What is the use of `updateOne()` and `updateMany()`?     | `updateOne()` updates only the first matching document; `updateMany()` updates all matching documents. |
| 9         | What is `upsert`?                                        | `upsert: true` creates a new document if no matching document exists.                                  |
| 10        | What is `save()` method used for?                        | `save()` inserts a new document or replaces an existing one if `_id` matches.                          |
| 11        | What is difference between deleteOne() and deleteMany()? | `deleteOne()` deletes one matching document, while `deleteMany()` deletes all matching documents.      |
| 12        | How to sort documents?                                   | Using `.sort({ fieldname: 1 })` for ascending or `-1` for descending order.                            |
| 13        | What is the default data format used in MongoDB?         | BSON (Binary JSON).                                                                                    |
| 14        | How is MongoDB different from SQL?                       | MongoDB is schema-less and stores data as JSON documents, while SQL uses fixed table schemas.          |
| 15        | What is use of `limit()`?                                | It limits the number of documents returned by a query.                                                 |
