Question -- Design and Implement following query using MongoDB  
                1. Create a collection called ‚Äògames‚Äô. 
                2. Add 5 games to the database. Give each document the following properties: name, gametype, rating (out of 100) 
                3. Write a query that returns all the games 
                4. Write a query that returns the 3 highest rated games. 
                5. Update your two favourite games to have two achievements called ‚ÄòGame 
                Master‚Äô and ‚ÄòSpeed Demon‚Äô. 
                6. Write a query that returns all the games that have both the ‚ÄòGame Maser‚Äô and 
                7. the ‚ÄòSpeed Demon‚Äô achievements. 
                8. Write a query that returns only games that have achievements. 

Solution -- 

Step 1Ô∏è‚É£: Create Database and Collection

use DYPIT

db.createCollection("games")

Step 2Ô∏è‚É£: Insert 5 Game Documents

db.games.insertMany([
  { name: "Call of Duty", gametype: "Action", rating: 95 },
  { name: "FIFA 23", gametype: "Sports", rating: 90 },
  { name: "Minecraft", gametype: "Adventure", rating: 92 },
  { name: "Candy Crush", gametype: "Puzzle", rating: 85 },
  { name: "Need for Speed", gametype: "Racing", rating: 88 }
])

Step 3Ô∏è‚É£: Query to Return All Games

db.games.find().pretty()

Step 4Ô∏è‚É£: Query to Return the 3 Highest Rated Games

db.games.find().sort({ rating: -1 }).limit(3).pretty()

Step 5Ô∏è‚É£: Update Two Favourite Games with Achievements

db.games.updateOne(
  { name: "Call of Duty" },
  { $set: { achievements: ["Game Master", "Speed Demon"] } }
)

db.games.updateOne(
  { name: "Minecraft" },
  { $set: { achievements: ["Game Master", "Speed Demon"] } }
)

Step 6Ô∏è‚É£: Query to Return All Games Having BOTH Achievements

db.games.find({
  achievements: { $all: ["Game Master", "Speed Demon"] }
}).pretty()

Step 7Ô∏è‚É£: Query to Return Games that Have Achievements

db.games.find({ achievements: { $exists: true } }).pretty()

Step 8Ô∏è‚É£: Verify All Data

db.games.find().pretty()
