                     Create collection Student with fields as Roll_No, Name, Class, Marks, Address, Enrolled_Courses. 
                    (Hint: One student can enrol in multiple courses. Use Array to store the names of courses enrolled)

üîç Queries

1Ô∏è‚É£ List the names of students who have enrolled in the course ‚ÄúDBMS‚Äù, ‚ÄúTOC‚Äù.


db.Student.find(
  { Enrolled_Courses: { $all: ["DBMS", "TOC"] } },
  { Name: 1, _id: 0 }
)

**Explanation:**
`$all` ensures both ‚ÄúDBMS‚Äù and ‚ÄúTOC‚Äù are present in the array.

---

2Ô∏è‚É£ List the Roll numbers and class of students who have marks more than 50 **or** class as TE.


db.Student.find(
  { $or: [ { Marks: { $gt: 50 } }, { Class: "TE" } ] },
  { Roll_No: 1, Class: 1, _id: 0 }
)

**Explanation:**
`$or` operator combines both conditions ‚Äî Marks > 50 **or** Class = TE.

3Ô∏è‚É£ Update the entire record of roll_no "A10".

db.Student.updateOne(
  { Roll_No: "A10" },
  { $set: { Name: "Arjun Patil", Class: "BE", Marks: 60, Address: "Pune", Enrolled_Courses: ["DBMS", "AI"] } }
)

**Explanation:**
`$set` replaces old field values with the new record data.

4Ô∏è‚É£ Display the names of students having 3rd and 4th highest marks.

db.Student.find({}, { Name: 1, Marks: 1, _id: 0 })
  .sort({ Marks: -1 })
  .skip(2)
  .limit(2)

**Explanation:**

* `sort({ Marks: -1 })` ‚Üí sorts in descending order.
* `skip(2)` ‚Üí skips top 2 (highest, 2nd highest).
* `limit(2)` ‚Üí shows next 2 (3rd and 4th highest).

5Ô∏è‚É£ Delete the records of students having marks less than 20.

db.Student.deleteMany({ Marks: { $lt: 20 } })

**Explanation:**
Deletes all students whose marks are below 20.

6Ô∏è‚É£ Delete only first record from the collection.

db.Student.deleteOne({})

**Explanation:**
`deleteOne({})` deletes **one random document**, typically the first one inserted.
