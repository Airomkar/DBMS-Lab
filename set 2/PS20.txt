## üíª **Step 1: Create Database and Table in MySQL**

Run these SQL commands in **phpMyAdmin** or MySQL terminal:

```sql

USE CollegeDB;

CREATE TABLE Student (
    Roll_No INT PRIMARY KEY,
    Name VARCHAR(50),
    Class VARCHAR(20),
    Marks INT
);

```

‚úÖ **Database ‚ÄúCollegeDB‚Äù created with table ‚ÄúStudent‚Äù and 4 records.**

---

## üíª **Step 2: Create PHP File for MySQL Connectivity**

**Filename:** `connect_mysql.php`

```php
<?php
// Step 1: Define database connection variables
$servername = "localhost";   // Server name
$username = "root";          // Default username in XAMPP/WAMP
$password = "";              // Default password is empty
$dbname = "CollegeDB";       // Database name

// Step 2: Create connection using mysqli
$conn = new mysqli($servername, $username, $password, $dbname);

// Step 3: Check connection
if ($conn->connect_error) {
    die("‚ùå Connection failed: " . $conn->connect_error);
}
echo "‚úÖ Connected successfully to MySQL database!<br><br>";

// Step 4: Execute SQL query
$sql = "SELECT * FROM Student";
$result = $conn->query($sql);

// Step 5: Display records if available
if ($result->num_rows > 0) {
    echo "<h3>üìÑ Student Table Data:</h3>";
    echo "<table border='1' cellpadding='5'>
            <tr><th>Roll No</th><th>Name</th><th>Class</th><th>Marks</th></tr>";
    
    // Output data of each row
    while ($row = $result->fetch_assoc()) {
        echo "<tr>
                <td>" . $row["Roll_No"] . "</td>
                <td>" . $row["Name"] . "</td>
                <td>" . $row["Class"] . "</td>
                <td>" . $row["Marks"] . "</td>
              </tr>";
    }
    echo "</table>";
} else {
    echo "No records found!";
}

// Step 6: Close the connection
$conn->close();
?>
```

---

## üíª **Step 3: Run the Program**

1. Save the file as **`connect_mysql.php`** in the folder:

   ```
   C:\xampp\htdocs\CollegeDB\
   ```
2. Start **Apache** and **MySQL** from the XAMPP Control Panel.
3. Open your browser and visit:

   ```
   http://localhost/CollegeDB/connect_mysql.php
   ```

## ‚úÖ **Explanation of the Code**

| Step  | Description                                                           |
| ----- | --------------------------------------------------------------------- |
| **1** | Database connection details (`localhost`, `root`, password, DB name). |
| **2** | Connection established using `new mysqli()`.                          |
| **3** | Checked for successful connection using `$conn->connect_error`.       |
| **4** | SQL query executed using `$conn->query($sql)`.                        |
| **5** | Fetched data using `$result->fetch_assoc()` in a loop.                |
| **6** | Closed the connection using `$conn->close()`.                         |

---

## ‚öôÔ∏è **Alternative Connection Method (PDO in PHP)**

You can also use **PHP Data Objects (PDO)** for a more secure connection:

```php
<?php
try {
    $conn = new PDO("mysql:host=localhost;dbname=CollegeDB", "root", "");
    $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "‚úÖ Connected successfully using PDO!";
} catch (PDOException $e) {
    echo "‚ùå Connection failed: " . $e->getMessage();
}
?>
```

# ‚úÖ **Summary of Key Functions**

| Function               | Description                                                                  |
